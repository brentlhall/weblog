About This Site

This site was built with Scroll. Breck Yunits quit working on it several months ago but it is still functional when installed to your terminal. There is a learning curve but I like the output.

I've simply downloaded all of his repositories to provide myself with the information I need use scroll.

If you download his breckyunits.com repository and build it using "build scroll" from your terminal, you'll have many examples for the scroll markup and you can get an idea how it looks by opening the index.html file in your browser.

Doing this with his other scroll based repositories gives you even more learning material.

Of course, you can download my repository from github.com/brentlhall/weblog/ and look at what I've built. Mine is much simpler. I've also put a template.txt in my folder to guide composing articles. Just remember to save your file as a scroll file before closing.

In a nutshell, I simply compose my articles using scroll markup in a folder you've chosen. Your articles and index file will end with ". scroll"

When you run "scroll build" from the terminal in your folder, it builds the html and text files.

I, then, upload the contents of this folder to my weblog github repository which I've setup to serve as my blog.

Having written this all out, it may sound like too much trouble but I assure you the product of your trouble is very satisfying. Plus, once you figure out the basics, it's much easier to keep building your blog.

Composing for the web in text or text with markup is so much easier.

I've not figured out how to add search, so, please use your device's built in search, "control+f" or "find", for example. Thanks! ****
